# Hello World Maven Project

This is my first Maven project with JColor for colored terminal output.

## How to Build
```bash
mvn clean package
```

## How to Run
```bash
java -jar target/hello-world-1.0-SNAPSHOT.jar
```

## Technologies Used
- Java 17
- Maven
- JColor library
- JUnit 5 for testing
